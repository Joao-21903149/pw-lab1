HTML5 e CSS

HTML 5

HTML5 (Hypertext Markup Language, versão 5) é uma linguagem de marcação para a WWW (World Wide Web) e é uma tecnologia chave da Internet, originalmente proposto por Opera Software. É a quinta versão da linguagem HTML.

Selectados Utilizado Website
style ✔️ Utilizei para meter o titulo de cada página assim como a foto da cidade
nav ✔️ Utilizei para fazer o menu
main ✔️ Utilizei para meter toda a informação de cada página
article ✔️ Utilizei nas fotos
section ✔️ Utilizei para dividir o codigo em sections e perceber onde começava e acabava cada informação
aside ❌ Não utilizei
footer ✔️ Utilizei no final da página para meter a minha informação

CSS

Cascading Style Sheets (CSS) é um mecanismo para adicionar estilo (cores, fontes, espaçamento) a um documento web.

Elementos Utilizado Website
universal ✔️ Utilizei para meter o background
tipo ✔️ Utilizei para meter o background azul
classe ✔️ Utilizei para meter toda a informação de cada página
id ✔️ Utilizei para meter o background azul
atributo ✔️ Utilizei nas fotos(target)
pseudo-classe ❌ Não utilizei
pseudo-elemento ✔️ Utilizei no poema
